# Simple TypeScript Form

This project is a simple HTML form that utilizes TypeScript for input validation. It includes a user-friendly interface and provides feedback based on user input.

## Project Structure

```
simple-ts-form
├── public
│   └── index.html        # HTML structure for the form
├── src
│   ├── index.ts          # TypeScript validation logic
│   └── styles.css        # CSS styles for the form
├── package.json           # npm configuration file
├── tsconfig.json          # TypeScript configuration file
├── .gitignore             # Git ignore file
└── README.md              # Project documentation
```

## Setup Instructions

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd simple-ts-form
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Build the project:**
   ```
   npm run build
   ```

4. **Open the form:**
   Open `public/index.html` in your web browser to view and interact with the form.

## Usage

- Fill in the input fields with the required data.
- Click the submit button to validate the input.
- Feedback will be provided based on the validation results.
